# Fotopedia 📸

A minimal and professional photography & video editing portfolio by Ashutosh.

---

## 🌟 Overview

Fotopedia is a personal portfolio website designed to showcase photography and video editing projects in a clean and modern layout.  

The platform highlights visual storytelling, refined editing, and a streamlined client booking experience.

## 🎯The goal of Fotopedia is simple:
- Present creative work in a professional format  
- Provide direct access to portfolio via Instagram  
- Enable clients to book slots easily  
- Maintain a clean and minimal brand identity

## 📷 About the Creator
Ashutosh is a photographer and video editor focused on capturing authentic moments and refining them into impactful visual stories.
Fotopedia represents a blend of creativity, precision, and clean visual presentation.
